# noale66.it

Sito per la cena di classe dei nati nel 1966 a Noale (VE), per festeggiare insieme i 60 anni. Pubblicato su GitHub Pages con dominio custom [noale66.it](https://noale66.it).

## Funzionalità

- Modulo di iscrizione con validazione e protezione anti-bot
- Backend su Google Sheets tramite Google Apps Script
- Dashboard admin protetta da password (modifica, cancellazione, export CSV)
- Condivisione e gruppo WhatsApp
- Design responsive mobile-first, stile New Wave anni '80
